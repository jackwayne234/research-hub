1. What Is an Equation?

The Balance Scale

An equation says: "the left side equals the right side." Think of it as a balance scale -- both sides must weigh the same to stay level.

The Golden Rule: Whatever you do to one side, you must do to the other side. Add 3 to the left? Add 3 to the right. The scale stays balanced.

2. One-Step Equations

Undo the Operation

A one-step equation needs exactly one inverse operation to isolate x. The four types are:

x + 5 = 12
0
x + 5 = 12
We need to get x alone. The 5 is added, so we subtract 5 from both sides.
1
x + 5 - 5 = 12 - 5
Subtract 5 from both sides.
2
x = 7
Done! x equals 7.
x - 3 = 8
0
x - 3 = 8
The 3 is subtracted, so we add 3 to both sides.
1
x - 3 + 3 = 8 + 3
Add 3 to both sides.
2
x = 11
Done! x equals 11.
4x = 20
0
4x = 20
x is multiplied by 4, so we divide both sides by 4.
1
4x / 4 = 20 / 4
Divide both sides by 4.
2
x = 5
Done! x equals 5.
x / 3 = 7
0
x / 3 = 7
x is divided by 3, so we multiply both sides by 3.
1
(x / 3) * 3 = 7 * 3
Multiply both sides by 3.
2
x = 21
Done! x equals 21.

3. Two-Step Equations

Unwinding in Reverse

Two-step equations need two operations. The trick: undo addition/subtraction first, then undo multiplication/division. Think of it as peeling off layers, outermost first.

The Order: Step 1 -- Undo any addition or subtraction (move the constant). Step 2 -- Undo any multiplication or division (isolate x).
3x + 5 = 20
0
3x + 5 = 20
Start. We need to isolate x. First, deal with the +5.
1
3x + 5 - 5 = 20 - 5
Subtract 5 from both sides to remove the constant.
→
3x = 15
Simplified. Now handle the multiplication by 3.
2
3x / 3 = 15 / 3
Divide both sides by 3.
✓
x = 5
Done! Check: 3(5) + 5 = 15 + 5 = 20. Correct!
2x - 7 = 9
0
2x - 7 = 9
Start. First, undo the -7.
1
2x - 7 + 7 = 9 + 7
Add 7 to both sides.
→
2x = 16
Now divide both sides by 2.
2
2x / 2 = 16 / 2
Divide both sides by 2.
✓
x = 8
Done! Check: 2(8) - 7 = 16 - 7 = 9. Correct!

4. Multi-Step Equations

The Full Process

Multi-step equations may require:

1. Distribute -- multiply through parentheses

2. Combine like terms -- simplify each side

3. Move variables -- get all x-terms on one side

4. Solve -- use two-step methods

Variables on Both Sides: When x appears on both sides, move all x-terms to one side and all constants to the other. The goal is always the same: isolate x.
2(x + 3) = 4x - 2
0
2(x + 3) = 4x - 2
Start. We have parentheses and variables on both sides.
1
2x + 6 = 4x - 2
Distribute the 2: 2 * x = 2x, 2 * 3 = 6.
2
2x - 2x + 6 = 4x - 2x - 2
Subtract 2x from both sides to group variables.
→
6 = 2x - 2
Simplified. Now solve the two-step equation.
3
6 + 2 = 2x - 2 + 2
Add 2 to both sides.
→
8 = 2x
Almost there. Divide both sides by 2.
✓
x = 4
Done! Check: 2(4+3) = 2(7) = 14. And 4(4)-2 = 16-2 = 14. Both sides match!
3(2x - 1) + 4 = 5x + 6
0
3(2x - 1) + 4 = 5x + 6
Start. Distribute first, then combine like terms.
1
6x - 3 + 4 = 5x + 6
Distribute: 3 * 2x = 6x, 3 * (-1) = -3.
2
6x + 1 = 5x + 6
Combine like terms on the left: -3 + 4 = 1.
3
6x - 5x + 1 = 5x - 5x + 6
Subtract 5x from both sides to group variables.
→
x + 1 = 6
Simplified. One more step.
✓
x = 5
Subtract 1 from both sides. Check: 3(2*5-1)+4 = 3(9)+4 = 31. And 5(5)+6 = 31. Correct!
